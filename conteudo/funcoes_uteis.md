# Funções Úteis

## Vetores
```
z = vector('character', length=6)          # vetores de caracteres
z
[1] "" "" "" "" "" ""

z = vector('logical', length=6)            # vetores lógicos
z
[1] FALSE FALSE FALSE FALSE FALSE FALSE    

z = vector('numeric', length=6)            # vetores de números
z
[1] 0 0 0 0 0 0
```

## Sequencias
```
seq(0,20,4)                                # sequencia em ordem crescente (inicio, fim, incremento)
[1]  0  4  8 12 16 20

seq(30,20,-4)                              # sequencia em ordem decrescente (inicio, fim, decremento)
[1] 30 26 22

seq(0,1.5,.25)                             # sequencia com incremento decimal
[1] 0.00 0.25 0.50 0.75 1.00 1.25 1.50
```

## Repetições
```
rep('A', 4)
[1] "A" "A" "A" "A"


```
