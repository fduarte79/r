# INTRODUÇÃO

## Aspectos Gerais

Estatística é uma coleção de métodos para planejar experimentos, obter dados e organiza-los, resumi-los, analisá-los, interpretá-los e deles extrair informações.

**Populaçao**: coleção completa de *todos* os elementos;\
**Censo**: coleção de dados relativos a *todos os elementos* de uma população;\
**Amostra**: subcoleção de *elementos extraídos* de uma população;\
**Parâmetros**: medida numérica que descreve uma característica de uma *população*;\
**Estatística**: medida numérica que descreve uma característica de uma *amostra*.

Obs: o que define se é um parâmetro ou estatística é a *população*.\
**100% da população** -> *parâmetro*\
100% da população -> **amostra (75%)** -> *estatística*
